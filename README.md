B3-Team modpack
======
This is the official repository and issue-tracker for the B3-Team Modpack<br>
Ceci est le référentiel officiel et le suivi des problèmes du ModPack B3-Team<br>

Need Help ? Besoin d’aide ?
======
When reporting an issue don’t forget to put the version of the ModPack in the title !<br>
Lorsque vous signalez un problème, n'oubliez pas d’indiquer la version du ModPack dans le titre !
